## Share Blog

Share-blog Is a website blog where people accross the world can share and discuss about the practices and cultures of each country. Built with ruby on rails

## Built With

- Ruby on rails
- SQL

### Prerequisites

- Ruby
- Rails
- PostgreSQL
- Git/Github
- VSCode or another equivalent tool
